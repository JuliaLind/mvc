matchRank()  : bool
Called if the hand does not already contain a pair and the rank of at least one of the cards in hand matches rank of a card in deck. Returns true if the rank in hand has a match either in the hand or in the deck. If there is a match in the hand it means that the hand contains two cards that contribute to two pairs rule, thus additionalValue attr is set to 2. If rank of dealt card instead matches in deck, it means only one card in hand contributes to the two pair rule, thus setting additionalValue to 1
